# 🌤️ OnFit — Weather-Aware Outfit Recommender

실시간 **온도·풍량·강수 확률** 데이터를 기반으로, 아침·점심·저녁별 최적의 레이어링(겹쳐 입기) 코디를 제안하는 웹·모바일 통합 서비스
> “밖에 나가서 후회할까?” 걱정을 없애주는 퍼스널 스타일 가이드.
